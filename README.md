
# Blog Frontend

Blog Built on Next js, Tailwind CSS, and Typescript. 
Live on [blog-arvind11](https://www.blog-arvind11.ml/) .

### Get the Code

- Clone Repo

```
mkdir blog_frontend
cd blog_frontend
git clone https://github.com/Arvind-4/Blog-Frontend.git .
```
- Install Dependencies

```
yarn install
```
