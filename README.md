### Get the Code

- Clone Repo

```bash
cd ~/Dev
mkdir ~/Dev/blog -p
cd ~/Dev/blog
git clone hhttps://github.com/Arvind-4/blog-frontend.git .
```

- Install Dependencies

```bash
pnpm install
```

- Make a `.env`

```bash
cp .env.sample .env
```

- Run dev server

```bash
pnpm dev
```
